#### Arduino_project

#Bluetooth controlled LCD display using arduino.

### Materials Required
1) Arduino UNO
2) LCD display(16*2)
3) HC05 Bluetooth module
4) Breadboard and jumper wires
5) USB cable


### Software Required
1) Arduino IDE
2) Arduino Automation App
